Прочитайте код приложения онлайн-банка, где пользователь может по желанию завести счета в трёх валютах. В программу забрался баг — вне зависимости от выбора пользователя сразу открываются все три счёта.
С помощью классов-обёрток и их свойств исправьте код так, чтобы открывались счета только в тех валютах, которые выберет пользователь. Счета необходимо создавать с нулевым балансом.
